# cljs-redux-demo

A small demo copying the Redux approach to a ClojureScript/figwheel setup

## Setup

- `lein dev`
- Open [http://localhost:3449/cards.html](http://localhost:3449/cards.html) for devcards, or
- Open [http://localhost:3449/index.html](http://localhost:3449/index.html) for the actual app
